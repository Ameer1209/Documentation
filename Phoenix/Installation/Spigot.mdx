1. Download the Retention Loader: You can find the latest version of Retention Loader on our [Shop](https://refinedev.xyz/resources/phoenix.3/).
2. After downloading Retention Loader, upload it on your server as a plugin in the plugins folder.
3. Once Retention is uploaded, start your server for the first time with Retention as the plugin.
4. Shut the server down.
5. Navigate to your plugins folder and find the "Phoenix" folder.
6. Open the global.yml file in the Phoenix plugin folder and fill in the database details.
7. Start the server up. If the database credentials are valid, phoenix should load right up.
