## Setting up a default arena

1. Upload an arena schematic to the WorldEdit/schematics folder
2. Use the command `/arena create <schematic>` to create the arena.
3. Use `/arena manage` and click on the created arena to access management options.
4. Teleport to the arena and set the spawns:
    - Spawn 1: Place a fence and a Zombie Head on it. (Required)
    - Spawn 2: Place a fence and a Creeper Head on it. (Required)
    - Spectator: Place a fence and a Skeleton Skull on it. (Optional)
1. Use `/arena manage` and save the schematic where the spawns are placed.
2. Use `/arena addkit <arena> <kit>` to add a kit to that arena.
3.  Use `/arena manage` to create copies of the arena.
4.  Use `/arena manage` to enable the arena.
